## 平台简介

露营营地预约管理系统是一套面向户外露营行业的综合服务平台，基于若依RuoYi-Vue框架二次开发，分为**用户前台**与**管理后台**两大模块，兼顾普通游客线上使用与运营人员后台管理需求。

* 前端采用Vue 2、Element UI，页面简洁美观，交互流畅。
* 后端采用Spring Boot、Spring Security、Redis & Jwt，架构稳定易维护。
* 沿用Jwt令牌认证机制，权限分级管控，保障系统数据安全。
* 分离用户首页与管理后台路由，首页免登录访问，后台严格权限校验。
* 整合营地预约、装备租赁、餐饮服务、露营攻略、订单管理等全业务功能。
* 支持动态菜单、角色权限、数据统计、日志记录等通用后台能力。

## 技术架构

本系统基于RuoYi-Vue标准技术栈进行业务拓展，前后端完全分离：

| 分类 | 技术选型 |
| :--- | :--- |
| 后端框架 | Spring Boot、Spring Security、MyBatis |
| 数据库 | MySQL |
| 缓存/认证 | Redis、Jwt |
| 前端框架 | Vue 2、Vue Router、Vuex |
| UI组件库 | Element UI |
| 辅助工具 | NProgress、Axios |

## 前台功能（用户端）
1.  系统首页：系统整体介绍、营地推荐、热门活动展示，支持免登录直接访问。
2.  营地展示：全量露营场地在线浏览，支持分类、筛选、查看场地详情。
3.  营地预约：在线选择场地、预约时段，完成露营场地预订。
4.  装备租赁：帐篷、桌椅、炊具等露营装备在线挑选与租赁。
5.  餐饮套餐：户外烧烤、简餐、下午茶等套餐查询与预定。
6.  露营攻略：户外知识、出行指南、安全提示等内容浏览。
7.  个人中心：用户信息管理、我的订单、我的预约、租赁记录查询。

## 后台功能（管理端）
### 一、露营业务管理
1.  营地管理：营地信息新增、编辑、上下架、场地状态维护。
2.  装备管理：露营装备分类、库存、价格、信息管理。
3.  餐饮管理：餐饮套餐创建、修改、状态管控。
4.  预约订单：所有营地预约订单查询、状态处理、记录追溯。
5.  租赁订单：装备租赁订单统一管理、订单进度跟踪。
6.  攻略管理：露营攻略文章发布、编辑、删除、排序。

### 二、系统基础功能（沿用若依原生）
1.  用户管理：平台注册用户信息查看、状态管理。
2.  部门管理：后台组织架构配置，树形结构展示。
3.  岗位管理：后台操作人员岗位配置。
4.  菜单管理：系统菜单、按钮权限标识配置。
5.  角色管理：角色权限分配、数据范围权限控制。
6.  字典管理：系统通用固定数据维护。
7.  参数管理：系统全局动态参数配置。
8.  通知公告：平台公告发布与维护。
9.  操作日志：后台人员操作行为记录与查询。
10. 登录日志：系统登录记录、异常登录统计。
11. 在线用户：实时监控当前在线后台账号。
12. 服务监控：服务器运行状态、内存、连接池监控。

## 运行环境
* 后端：JDK 8+、MySQL 5.7+ / 8.0、Redis
* 前端：Node.js 14+
* 部署环境：Windows / Linux
